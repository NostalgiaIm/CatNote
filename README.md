喵喵便签

简介

喵喵便签是一款适用于 Windows x64 的轻量级便签工具，支持多主题、搜索、置顶和快捷操作。

目前尚未推出桌面端（指移动端或其他平台暂未支持）。

---

功能

· 搜索
    通过顶部搜索栏，可按内容或修改时间快速筛选便签。
· 自由拖动
    所有便签均可随意拖拽至屏幕任意位置。
· 一键整理
    在 Edit 菜单中选择 “整理”，所有便签自动对齐排列。
· 主题切换
  · 全局主题：淡红、淡蓝、淡绿、淡粉、黑白、原色等。
  · 单卡主题：每张便签可独立修改配色。
· 添加到桌面
    点击卡片上的 “添加到桌面”，即复制一份相同便签到系统桌面。
· 置顶
    开启后便签始终显示在最上层，不被其他窗口遮挡。
· 全局快捷键
    程序后台运行时，按下 Ctrl + Q 可快速创建新便签。

---

系统要求

· Windows 10 / Windows 11（64 位）
· 无需安装，解压即可使用

---

启动方式

1. 下载并解压项目压缩包。
2. 进入解压后的目录，双击运行 start.bat（或对应的批处理文件）。
3. 程序启动后显示主界面，并驻留系统托盘。

---

使用说明

操作 方法
新建便签 主界面点击 + 按钮，或后台按 Ctrl + Q
搜索 在顶部搜索框输入关键词
拖动便签 按住便签标题栏拖动
修改卡片主题 点击卡片上的调色板按钮
切换全局主题 主菜单 → 主题
一键整理 Edit → 整理
置顶 卡片右键菜单 → 置顶
复制到桌面 卡片右键菜单 → 添加到桌面
